🧩 Sudoku Mestre

Um jogo de Sudoku interativo desenvolvido para desafiar o raciocínio lógico e proporcionar uma experiência dinâmica ao usuário. O projeto foi construído com foco em performance, organização de código e usabilidade.

🚀 Demonstração

🔗 Acesse o jogo:
((https://gersonpedroso.github.io/Sudoku-master/))

🛠️ Tecnologias utilizadas

Este projeto foi desenvolvido utilizando:

Node.js
TypeScript
HTML5
CSS3
JavaScript

💡 Aprendizados

Durante o desenvolvimento deste projeto, foram aprimoradas habilidades como:

Lógica de programação
Estruturação de algoritmos
Manipulação de estados
Organização de código com boas práticas
Desenvolvimento de interfaces interativas
